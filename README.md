# EcoOil Backend 🚀

This is a Spring Boot backend project for managing oil requests and users.

## Tech Stack
- Java (Spring Boot)
- Maven
- MySQL
- Docker

## Features
- User Management
- Oil Request Management
- REST APIs
- Login System

## Run Locally

```bash
./mvnw clean package
java -jar target/ecooil-backend-0.0.1-SNAPSHOT.jar
